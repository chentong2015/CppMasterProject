TODO. C++ 采用使用头文件来包含声明的约定
- 在一个头文件中进行声明，在每个.cpp文件或其他需要该声明的头文件中引入
- 使用#include指令在编译之前将头文件的副本直接插入.cpp文件中

C++ Header File
- 定义声明的约定和函数原型，也可在其中定义函数的实现
- 在引入头文件时，它将自动找到对应的cpp文件(等效于被隐式地引入)
- 在引入头文件时，它将自动引入它引入的父类header

C++ Source File
- 定义header头文件类型的函数实现
- 定义其他单独的函数 => 不推荐，应该封装到class类中
- 源文件被直接引入时等效于copy文件中定义的全部函数
