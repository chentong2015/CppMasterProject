C++ Header File 头文件:
- 定义函数原型(被其他文件include引入)，可在其中定义函数的实现
- 在引入头文件时，它将自动找到对应的cpp文件(等效于被隐式地引入)
- 在引入头文件时，它将自动引入它引入的父类header

C++ Source File 源文件:
- 定义header头文件类型的函数实现
- 定义其他单独的函数 => 不推荐，应该封装到class类中
- 源文件被直接引入时等效于copy文件中定义的全部函数
