TODO. C++ 如何编译和运行
1. gcc编译源码文件再执行
   - gcc is the C compiler frontend to GCC.
   - g++ is the C++ compiler frontend to GCC.

   编译执行的cpp源文件并执行
   > brew install gcc
   > g++ -o hello hello.cpp
   > ./hello
   
2. Makefiles批量编译源代码, 只包含一个可执行的main方法
   - /usr/bin/make
   - cmake build systems 系统的构建工具
